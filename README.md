# Unlock Bot

Unlock Bot — это Telegram-бот, предназначенный для автоматической разблокировки учетных записей пользователей в Active Directory.

## Что он делает?

Этот бот позволяет пользователям самостоятельно разблокировать свои учетные записи AD, которые были заблокированы, например, из-за многократного ввода неверного пароля. Это снижает нагрузку на службу технической поддержки и ускоряет процесс восстановления доступа для пользователей.

## Функционал

-   **Регистрация пользователей:** Новые пользователи должны пройти процедуру регистрации, отправив свою рабочую почту. Запрос на регистрацию подтверждается администратором.
-   **Разблокировка учетной записи:** Зарегистрированные пользователи могут разблокировать свою привязанную учетную запись AD.
-   **Привязка учетной записи:** Пользователи могут привязать свой Telegram-аккаунт к своей учетной записи AD.
-   **Уведомления для администратора:** Бот отправляет администраторам уведомления о запросах на регистрацию и других важных событиях.
-   **Просмотр заблокированных учеток (для администраторов):** Администраторы могут получить список всех заблокированных в данный момент учетных записей.

## Команды

-   `/start` или `/reg` - Начать процесс регистрации для новых пользователей.
-   `/unlock` - Разблокировать свою учетную запись Active Directory.
-   `/connect` - Привязать свою учетную запись AD к вашему Telegram-аккаунту. Можно использовать как `/connect <логин>`, так и просто `/connect` для интерактивного режима.
-   `/status` - **(Только для администраторов)** Получить список заблокированных в данный момент пользователей.

## Как это работает

Проект состоит из трех основных частей:

1.  **Telegram Бот:** Написан с использованием библиотеки `aiogram`. Он обрабатывает команды от пользователей и взаимодействует с остальными компонентами.
2.  **Бэкенд Active Directory:** Использует `pyad` для взаимодействия с Active Directory (проверка существования пользователя, разблокировка). Для получения списка заблокированных пользователей используется PowerShell.
3.  **База данных:** SQLite с `SQLAlchemy` для хранения информации о зарегистрированных пользователях, их правах и привязке к учетным записям AD.

## Установка и настройка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <URL репозитория>
    cd unlock_bot
    ```

2.  **Установите зависимости:**
    Рекомендуется использовать виртуальное окружение.
    ```bash
    python -m venv venv
    source venv/bin/activate  # для Linux/macOS
    # или
    venv\Scripts\activate  # для Windows
    
    pip install -r requirements.txt 
    # Или если вы используете uv:
    uv pip install -r requirements.txt
    ```

3.  **Настройте переменные окружения:**
    Создайте файл `.env` в корневом каталоге проекта и добавьте в него следующие переменные:

    ```env
    BOT_TOKEN="ВАШ_ТЕЛЕГРАМ_БОТ_ТОКЕН"
    DOMAINS="domain1.com;domain2.local"
    SEARCH_ZONE="OU=Users,DC=yourdomain,DC=com"
    ADMIN_CHAT="ID_АДМИН_ЧАТА_В_ТЕЛЕГРАМ"
    DATABASE_PATH="путь/к/вашей/базе_данных.db"
    ```
    - `BOT_TOKEN`: Токен вашего Telegram-бота.
    - `DOMAINS`: Список доменов AD через точку с запятой.
    - `SEARCH_ZONE`: Область поиска в AD (Distinguished Name).
    - `ADMIN_CHAT`: Числовой ID чата или пользователя в Telegram, куда будут приходить запросы на администрирование.
    - `DATABASE_PATH`: Путь к файлу базы данных SQLite.

4.  **Запустите бота:**
    ```bash
    python -m unlock_bot.scripts.start
    ```

## Для администраторов

Администраторы управляют регистрацией новых пользователей. Когда новый пользователь отправляет команду `/start`, он указывает свою почту. Если пользователь с такой почтой существует в AD, в чат, указанный в `ADMIN_CHAT`, приходит сообщение с кнопками "Да" и "Нет".

-   **Да:** Пользователь регистрируется в боте, и ему отправляется уведомление об успешной регистрации.
-   **Нет:** Пользователю отказывают в регистрации.

Команда `/status` позволяет администраторам в любой момент увидеть список заблокированных учетных записей в AD.
